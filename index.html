Standardize Part Number


Simple UI to demonstrate a use of XSB's Partnumber Standardizer web api



Suppliers do not provide item identifying information, such as manufacturer names and part numbers, in a consistent way. This lack of standardization between vendors and data sources makes it difficult to find identical parts across the supply chain, assess pricing, regulatory conformance, and other important procurement related activities. XSB's partnumber-standardizer web api standardizes part numbers so identical parts can be found.


The part number standardizer web api standardizes part numbers removing dashes and normalizing to upper case.
